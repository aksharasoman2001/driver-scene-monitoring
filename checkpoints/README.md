# Checkpoints

Model weights are not stored in this repository due to file size limits.

Download all ONNX models from the [Releases page](../../releases) and place them here.

## Expected Files

```
checkpoints/
├── dms_eye.onnx                ← 6.11 MB  — DMS eye classifier
├── lane_detection_final.onnx   ← 258.5 MB — Lane detection (FP16)
└── yolo_detection.onnx         ← 10.6 MB  — Object detection (10 classes)
```
